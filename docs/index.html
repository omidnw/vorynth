✦ Features

📡

Multi-Source Collection

RSS feeds with per-source fetch windows and rate limiting. 13 seed sources covering AI, engineering, security & more.

🧠

LangGraph Intelligence

AI workflow: collect → normalize → dedup → rank → classify → analyze → brief. All local, all private.

🤖

4 LLM Providers

OpenAI, Anthropic Claude, Google Gemini, Groq — switch per query or per generate. Bring your own key.

🔍

FTS5 Search + AI RAG

Blazing-fast full-text search with Persian/Arabic support. Ask questions in natural language with cited answers.

🌐

i18n + RTL

English & Persian UI with RTL auto-detection for AI output. More languages via standard ICU message format.

🔒

Privacy First

No cloud account. No telemetry. Your API keys, history, and insights stay under your control — local-first.

💾

Backup & Restore

Full database snapshots with manifest. One-click export, restore, and delete from the settings panel.

🌙

Dark Mode + Themes

System-aware theme with multiple accent colors. Beautiful design that adapts to your preference.
